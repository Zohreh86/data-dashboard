# Saarland Data Dashboard

A data analysis project using real German statistics from Destatis.

## What it does
- Analyzes Saarland population data from 2015–2023
- Calculates growth, average, and peak year
- Generates a annotated line chart saved as PNG

## Technologies used
- Python 3
- Pandas
- Matplotlib

## How to run

1. Install dependencies:
   pip3 install pandas matplotlib

2. Run the script:
   python3 main.py
