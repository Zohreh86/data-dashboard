# Saarland IT Job Market Dashboard

A data visualization project analyzing IT job posting trends in Saarland, Germany.

## What it does
- Reads monthly IT job posting data
- Calculates statistics (average, min, max)
- Generates a line chart saved as PNG

## Technologies used
- Python 3
- Matplotlib

## How to run

1. Install dependencies:
   pip3 install matplotlib

2. Run the script:
   python3 main.py

## Output
A chart showing IT job posting trends across 8 months.
